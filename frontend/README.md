# AgriPower Frontend

React + Vite application for the AgriPower Electric Power Distribution System.

> Created by **Sharook**

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Recharts (charts)
- Axios (API calls)
- React Router DOM

## Development

```bash
npm install
npm run dev
```

Runs on http://localhost:3000

## Build

```bash
npm run build
```

Output goes to `dist/` folder.
